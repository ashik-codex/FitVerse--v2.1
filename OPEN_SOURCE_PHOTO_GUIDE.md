# Open-source exercise photo guide

FitVerse v1.6 replaces the generated/stick-style movement visuals with source photographs from the Free Exercise DB project.

Source: https://github.com/yuhonas/free-exercise-db
License: Unlicense / public-domain dataset.

Each exercise displays a start-position and working-position photograph when the corresponding source entry is available. Written setup, movement, finish and return steps remain visible for clarity. The source link is shown inside every exercise guide.
