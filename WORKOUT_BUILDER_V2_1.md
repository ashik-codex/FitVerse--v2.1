# FitVerse v2.1 — Workout Builder Upgrade

The Create Workout flow now supports:

- One body part workouts
- Two body part workouts
- Push / Pull / Legs sessions
- Full body sessions
- Fully custom sessions
- Exact exercise-count selection: 4, 5, 6, 7 or 8 exercises
- Automatic recommended workout generation
- Search by exercise name, muscle, equipment, goal and description
- One-click search clearing and full filter reset
- Category chips and category dropdown
- Equipment, difficulty, location and exercise-type filters
- Selected-only and matching-focus-only filters
- Offline exercise photo thumbnails in the exercise picker
- Maximum exercise-count protection
- Selected exercise review, removal and reordering
- Dynamic focus labels and plan metadata

All existing offline exercise photos, replacements, live workout, nutrition, analytics and progress features remain available.
